O FindWater é um sistemas de monitoramento de redes de abastecimento de água, que ajudará você a detectar vazamentos.

Invista em tecnologia de detecção de vazamentos em redes de distribuição!

Integração hardware + software

Sensoriamento para monitoramento dos nívies de pressão na malha de distribuição, correlacinado com os níveis de umidade do solo, integrando o software de análise de dados e tomada de decisão baseada e inteligência artificial

Dispositivo físico para produção

Aplicação de sensores de medição para monitoramento de pressão e medidores de resistividade do solo para a detecção afertiva da existência de vazamentos.

Software de processamento de dados

Software que permite o processamento das informações coletadas a partir de técnicas estatísticas e de inteligência artificial para que seja possível obter um relatório detalhado do sistema de saneamento.

Acompanhamento em tempo real

O FindWater é um sistema na nuvem que permite o acompanhamento dos seus dados de saneamento em tempo real através da internet.
